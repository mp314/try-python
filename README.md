# try-python

> My Python snippets and boilerplates.

## Getting Started

### Prerequisites

- [Python 3](https://www.python.org/downloads/)

### Running

```shell
# start virtual enviroment and install requirements
python3 -m venv .venv
. .venv/bin/activate
pip3 install <any requirements>

# mypackage
cd try-package
python3 -m mypackage # or ./run-mypackage.sh
```
